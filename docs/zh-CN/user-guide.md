# 用户指南

[English](../en/user-guide.md) · [项目主页](../../README.zh-CN.md)

## 基本概念

每个网格对象分别拥有面、边、点三套独立图层。一个网格元素可以同时属于多个图层。
当前元素类型决定面板显示哪套图层及对应操作。

标注分配在编辑模式中修改。图层颜色和分配关系随对象保存在 `.blend` 文件中，
不会占用材质或顶点颜色。

## 基础流程

1. 选择网格，按 `N` 打开侧边栏，进入 **Mesh Annotation（网格标注）**。
2. 进入编辑模式，选择面、边或点选择模式。
3. 点击 **+** 新建图层，然后命名并设置颜色。
4. 选择网格元素，执行**分配选中**。
5. 使用图层眼睛按钮控制单层显示，或用**独显活动层**隔离查看。

## 分配工具

- **分配选中**：将当前选区加入活动图层。
- **分配循环**：从选区推导完整的面循环、边循环或点路径后再分配。
- **选中/循环 → 新图层**：一次完成新建与分配。
- **分配到已有图层**：无需先切换活动层。
- 点模式中的**连接数**：查找拥有指定相邻边数量的顶点。

如果循环推导存在歧义，请调整选区后重试；插件不会静默选择任意路径。

## 选择与清理

- **选择图层元素**：选中某个图层中的全部元素。
- **从选区激活**：激活当前选区中占比最高的图层。
- **移除选中**：从指定图层移除当前选区。
- **清理选中**：可只移除顶层分配，也可移除选中元素的全部分配。
- 删除图层时会同时清除该图层的分配关系。

## UV 缝工作流

在面模式中，使用**标记活动图层缝**将活动面图层的边界标记为 UV 缝；
**标记全部图层缝**会处理所有面图层。该操作不会改变网格几何。

## 显示设置

- **显示叠加**是总开关。
- 图层眼睛按钮与**独显活动层**控制绘制范围。
- **透明度、线宽、点大小**控制视觉权重。
- 面、边、点独立偏移可减少深度冲突。
- **边截断**会缩短彩色边线的两端。
- **穿透显示**会改变深度测试，使背面的标注也可见。

偏移只需达到消除闪烁的程度；数值过大会让标注看起来脱离表面。

## 实用图层方案

- **拓扑检查：**极点、夹痕、高密度区域、待清理区域。
- **重新拓扑：**已完成区域、不确定边流、关键变形循环。
- **UV 规划：**用面图层规划岛，再将图层边界转换为缝。
- **团队备注：**使用稳定名称，例如 `review`、`approved`、`rework`。

## 语言

插件偏好设置提供**自动、英文、中文**。自动模式在受支持的中文 Blender 区域设置下
使用中文，其他情况使用英文。

## 数据与限制

- 数据按对象存储，并随 `.blend` 文件保存。
- 除非其他工具专门读取插件自定义数据，否则标注不会影响 FBX/OBJ 导出。
- 编辑一个对象不会修改其他对象的图层。
- 默认不绑定快捷键，可在 Blender 键位映射中绑定对应操作。
- 高密度评估网格需要更多绘制成本。交互变慢时可隐藏不用的图层，并减少穿透显示。
